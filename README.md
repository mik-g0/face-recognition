# Face Recognition System

## Описание проекта
Система биометрического распознавания лиц с использованием **MTCNN** и **InceptionResnetV1** для идентификации личности по изображениям.

**Функции:**
- Распознавание лица (**Identify**)  
- Добавление нового пользователя в базу (**Enroll**)  
- Управление базой пользователей (**DB Manager**)  

---

## Требования
- Python 3.8+  
- Виртуальное окружение (рекомендуется)  
- Установленные зависимости из `requirements.txt`  

---

## Установка

1. Скачайте и распакуйте проект.  
2. Создайте виртуальное окружение:
```bash
python -m venv face_env
```

3. Активируйте окружение

Установите зависимости:
```bash
pip install -r requirements.txt
```

---

## Запуск приложения

В корне проекта выполните:
```bash
streamlit run app_streamlit.py
```

В браузере откроется интерфейс с тремя вкладками:

- Identify — распознавание лиц

- Enroll — добавление новых пользователей

- DB Manager — управление базой

---

## Работа с базой данных

В проекте уже есть база face_db.pkl и миниатюры в db_thumbs/.

Пользователю не нужно выполнять preprocess или обучение.

Новых пользователей можно добавлять только через вкладку Enroll.
